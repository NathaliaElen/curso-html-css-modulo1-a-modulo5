Principais Formatações

Negrito / Destaque

Nessa frase temos um termo em negrito usando a tag B. Essa tag não é semântica.

Nessa frase temos um termo em destaque usando a tag STRONG. Essa tag é semântica.

Itálico / Ênfase

Nessa frase temos um termo em itálico usando a tag I. Essa tag não é semântica.

Nessa frase temos um termo com ênfase usando a tag EM. Essa tag é semântica.

Texto marcado

Podemos criar também um texto marcado usando a tag MARK.

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno.

A Tag BIG está obsoleta, a Tag SMALL não.

Texto excluído

Podemos marcar um texto como excluído usando a tag DEL para indicar que ele deve ser lido, mas não considerado.

Existe também o tachado usando a tag S, não é semântica.

Texto inserido

Podemos marcar um texto como inserido usando a tag INS para dar ênfase e indicar que ele foi inserido depois.

Existe também o sublinhado usando a tag U, não é semântica.
